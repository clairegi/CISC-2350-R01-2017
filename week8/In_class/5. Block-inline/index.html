This is an h1 element. Since it's a block-level element, it takes up all the space from left to right - see the red border.



This is a paragraph element. It's also a block-level element, so takes up all the space from left to right as well (see blue border).



This is a div inside a div. Div is a block-level element and it will also stretch to take up the width of its parent element if no width is specified. See orange border.


Unlike divs, span elements are inline elements. This means they take up only as much width as necessary. You can see this by the green outline around it. This makes span elements great for styling text within other block elements such as paragraphs. I want to write a paragraph with pink text, purple text, and blue text.
